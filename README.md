
#  CSS Full Course

A comprehensive collection of CSS concepts and practical examples created while learning Cascading Style Sheets (CSS). This repository covers the fundamentals as well as advanced CSS topics through simple and easy-to-understand examples.

---

##  About

This repository contains hands-on examples of various CSS properties and techniques. Each topic is organized into separate folders with HTML files for better understanding and practice.

---

## Topics Covered

-  CSS Colors
-  Background Properties
-  Box Model
-  Height & Width
-  Margin & Padding
-  Border Properties
-  Visibility Property
-  Image Properties
-  Units (Absolute Units)
-  Pseudo Classes
-  CSS Transitions
-  CSS Animations
-  CSS Transform
  - Rotate
  - Scale
  - Skew
-  More CSS Practice Examples

---


##  Technologies Used

- HTML5
- CSS3

##  Learning Objectives

- Understand CSS fundamentals
- Practice styling web pages
- Learn CSS layout techniques
- Explore animations and transitions
- Master CSS transform properties
